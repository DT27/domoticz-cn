# Domoticz Docker版 中文

代码与原版一样，无改动

设置默认语言为中文

网页登录帐号密码均为`domoticz`，请在设置中更改

数据库文件、ssl证书文件、日志文件及博联万能遥控配置文件目录均为`/config`

默认http端口`8080`，默认https端口`443`
